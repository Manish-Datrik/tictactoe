Tic Tac Toe Game https://manish-datrik.github.io/tictactoe/

I built this game as a simple digital version of the classic Tic Tac Toe game. It is an interactive browser game where players place X and O on a 3×3 grid and try to make a straight line before their opponent.

How to play

Two Player Mode:
Two players take turns placing their marks on the board. The first player to align three marks horizontally, vertically, or diagonally wins.

Bot Mode:
Play against the computer. The player starts first, and the bot makes its move automatically.

The Goal

The goal is to place three matching symbols in a row before the opponent. If all squares are filled without a winner, the game ends in a draw.

Why I made this

I created this project to practice game logic, turn-based gameplay, and basic AI using JavaScript.
